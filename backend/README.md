## Description

The Backend for DAO Messaging Platform. This server scrapes data [karmahq.xyz](https://www.karmahq.xyz/). In Development ...!

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
